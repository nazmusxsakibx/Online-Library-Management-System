# 📚 Online Library Management System

A web-based Library Management System built with JavaScript that allows users to manage books, members, and library operations efficiently.

---


## 🚀 Features

- 📖 Add, update, and delete books
- 👤 Manage library members
- 🔍 Search and filter books
- 📋 Track book availability (issued / available)
- 🗂️ Organized and clean UI for easy navigation

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| JavaScript | Core programming language |
| HTML | Structure and layout |
| CSS | Styling and design |

---

## 📁 Project Structure

```
Online-Library-Management-System/
│
└── Library-Management-System-main/
    ├── index.html        # Main entry point
    ├── style.css         # Styling
    └── script.js         # Core logic
```

---

## ⚙️ How to Run

1. **Clone or download** this repository
2. Open the `Library-Management-System-main` folder
3. Double click on **`index.html`**
4. The app will open in your browser — no server needed!
